Cool button, Bro.

A nice button that promises to call you back.

It takes your callback(s), wraps them in promises using the $q library, and then nicely disables and enables the button while changing the labels for you.

Example: saves in {{timeoutSeconds.number}} milliseconds

Maybe your button calls and API and you don't know how long it will take to respond.
Change the Time to reset the button after you save.

Example: No "saved" state

By default Cool Button uses a "saved" state, which prevents clicking unless a target has been marked "dirty". Disabling the "saved" state lets the user click the button any time except during the "saving" state.
